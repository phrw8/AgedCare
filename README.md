# Aged Care
O Aged Care é um sistema desenvolvido para auxiliar na contratação de técnicos em enfermagem utilizado para a apresentação do meu trabalho de TCC(Trabalho de conclusão de curso), conectando profissionais da área com usuários que precisam de seus serviços.
